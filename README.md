Aegis is a premium, state-of-the-art Flutter fitness chat application. It demonstrates the power of **GenUI** (Generative User Interfaces) paired with **Gemini GenAI** and **Riverpod** state management, providing a personalized conversational coaching experience that renders fully interactive, reactive, and stateful widgets directly in the chat window.

---

## рџЊџ Key Features

*   **GenUI Architecture:** Translates structured JSON responses from the Gemini API into reactive, functional Flutter widgets inline.
*   **Dynamic Isolated State (Riverpod):** Each generated widget is bound to an isolated, dynamic family state provider (`widgetStateNotifierProvider(widgetId)`). This ensures:
    *   **Independence:** Interacting with one log bubble (e.g., ticking off an exercise) does not affect other logs in the history.
    *   **Persistence:** Widget states survive scroll-recycling when they move off-screen and back.
*   **Aegis AI Persona:** Instructed via system prompts to act as a supportive fitness and nutrition coach.
*   **Local Sandbox Simulation:** Fallback engine simulates AI widget generation when no Gemini API key is configured.
*   **Premium Dark Aesthetic:** Violet and Teal brand accents, custom glassmorphism overlays, and smooth micro-animations.

### рџ“‹ Prerequisites

Ensure you have Flutter and Dart installed:

*(This project was built and validated on Flutter 3.44.0 and Dart 3.12.0).*

### рџљЂ Running the App

1.  **Clone / Navigate to the project directory:**
    ```bash
    cd fitness_genui_chat
    ```

2.  **Fetch dependencies:**
    ```bash
    flutter pub get
    ```

3.  **Run build runner (to compile Riverpod annotations):**
    ```bash
    dart run build_runner build --delete-conflicting-outputs
    ```

4.  **Run the application:**
    ```bash
    flutter run
    ```

## рџ”‘ Configure Gemini API Key

1.  Click the **Gear Icon** in the top-right corner of the chat toolbar.
2.  Input your **Gemini API Key** from [Google AI Studio](https://aistudio.google.com/).
